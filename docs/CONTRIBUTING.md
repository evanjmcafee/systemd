# Contributing

Thank you so much for considering contributing to Liberated `systemd`!

If you wish to assist with Liberated `systemd`, the I would welcome help with regards to testing and automation. See https://github.com/Jeffrey-Sardina/systemd-suite, which is where I manage testing and automation code. I will be updating that repo with areas I am working on in which assistance from other developers would be welcomed.

If you want to contribute to features, bug fixes, optimizations, security enhancements, etc of `systemd`, please contribute to the base `systemd` repo: https://github.com/systemd/systemd?tab=contributing-ov-file
